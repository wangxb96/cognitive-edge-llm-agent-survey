## Category Coverage | 分类覆盖范围

- A Foundations: edge intelligence baseline and LLM fundamentals
- B Challenges: resource bottlenecks, latency/stability, privacy/security
- C Optimization: data/context, compression, serving/runtime, edge-cloud collaboration
- D Agentic: tool-use loops, multi-agent coordination, SLM-first deployment
- E Applications: mobility, healthcare, industrial and consumer on-device AI
- F Evaluation & Trust: reproducibility metrics and governance/security baseline

## Maintenance Rules | 维护规则

- Add new papers by category first, then update cross-category links.
- 新增文献先按分类归档，再更新跨分类关联。

- Keep hardware and evaluation settings explicit for each empirical claim.
- 每条实验结论必须注明硬件与评测设置。

- Prefer Pareto-style reporting (quality-latency-energy) over single-metric claims.
- 优先使用质量-时延-能耗 Pareto 报告，避免单指标结论。
