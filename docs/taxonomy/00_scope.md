# Scope and Positioning | 范围与定位

## EN

This survey taxonomy focuses on cognitive edge computing for LLMs and AI agents under strict resource constraints:

- Compute budget
- Memory and KV-cache budget
- Latency and tail-latency budget
- Energy/thermal budget
- Privacy/security governance constraints

Primary objective: preserve reasoning and agent capability under deployment budget, rather than optimizing a single metric.

## 中文

本综述分类聚焦资源受限条件下的大模型与智能体边缘部署：

- 算力预算
- 内存与 KV-cache 预算
- 时延与尾时延预算
- 能耗与热预算
- 隐私、安全与治理约束

核心目标：在部署预算内保持推理能力与智能体能力，而非追求单一指标最优。
